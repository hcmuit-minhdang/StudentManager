
# Student Management System (C++ OOP)

A simple C++ program for managing students using Object-Oriented Programming (OOP).  
Allows adding, viewing, searching, removing, and sorting students by GPA.

## Features
- Add new students
- Show all students
- Find student by ID
- Remove student by ID
- Sort students by GPA (descending)

## Build & Run

Compile and run using g++:

```bash
g++ main.cpp Student.cpp StudentManager.cpp -o StudentManager
./StudentManager
```
